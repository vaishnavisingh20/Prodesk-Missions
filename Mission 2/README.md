## Advanced Budget Tracker
---
The Advanced Budget Tracker is a dynamic web application that helps users manage their finances efficiently.

It allows users to track expenses, monitor remaining balance, visualize spending through charts, and export reports as PDFs — all in a clean and responsive interface.

---
## Features
- Set Salary & Track Expenses
- Real-time Budget Visualization (Pie Chart)
- Local Storage Support (data persists after refresh)
- Multi-Currency Selection (USD, EUR, INR, GBP)
- Low Balance Alert System
- Download Budget Report (PDF)
- Delete Individual Expenses
- Responsive Design
---
## Tech Stack
- HTML5 - Structure
- CSS3 - Styling and layout
- Javascript - Logic and interactivity
- Chart.js - Data Visualization
- jsPDF - PDF report generation
- FontAwesome - Icons
---
## Key Functionalities
- Budget Visualization
- Data Persistence
- Smart Alert System
- PDF Export
- Currency Support
