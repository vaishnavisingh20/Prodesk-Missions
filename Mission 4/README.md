## AI Cover Letter Generator
---
The AI Cover Letter Generator is a web application that creates professional cover letters instantly based on user input.

It simulates an AI writing assistant by combining user-provided details into a well-structured, ready-to-use cover letter — complete with a smooth UI and interactive feedback.

---
## Features
- Smart Template-Based AI Generation
- Loading Simulation (AI Thinking Effect)
- Input fields:
   Name
   Job Role
   Company Name
   Key Skills
- Instant Letter Generation
- Copy to Clipboard Button
- Scrollable Output Box for Long Letters
- Responsive & Clean UI Design
---
## Tech Stack
- HTML5 - Structure
- CSS3 - Styling and layout
- Javascript - Logic and interactivity
---
## How it works
Letter Generation Flow
- User fills the form
- Clicks Generate Cover Letter
- App:
1. Shows loading state ("Generating...")
2. Waits 2 seconds (AI simulation)
3. Injects data into a structured template
- Output is displayed in a formatted box
