## AI Cover Letter Generator
The AI Cover Letter Generator is a simple web application that helps users quickly create a professional cover letter by entering basic information such as their name, job role, company name, and skills.

The application dynamically generates a formatted cover letter using JavaScript and allows users to copy it directly to the clipboard.

---

## Technologies Used
- HTML5 – Structure of the application
- CSS3 – Styling and layout
- JavaScript (ES6) – Application logic and dynamic content generation
---

## Project Structure
- index.html
- script.js
- styles.css
- README.MD
- Note: There is no Prompts.md
---

## How to run the project
- Step 1: Clone or download the repository
- Step 2: Open the project folder
- Step 3: Open index.html in your browser
---

## Example workflow
- Enter details
- Click on Generate Cover Letter
- The appliaction will generate a formatted cover letter automatically
