## Digital Agency Portfolio
---
This project is a responsive digital agency landing page built using HTML, CSS, and JavaScript.

It showcases a modern UI with features like dark mode toggle 🌙, responsive navigation 🍔, and interactive service cards ✨ — designed to deliver a smooth user experience across devices.

---
## Features
- Modern UI Design with clean layout
- Dark / Light Mode Toggle
- Fully Responsive (Mobile + Desktop)
- Hamburger Menu for smaller screens
- Sticky Navigation Bar
- Smooth UI Transitions
- Hover Effects & Animations
- Service Cards with Lift Effect
---
## Tech Stack
- HTML5 - Structure
- CSS3 - Designing, Styling
- Javascript - Interactivity
---
## Key Functionalities
- Dark Mode Toggle
 1. Switch between light and dark themes
 2. Dynamically updates UI colors using CSS variables
- Responsive Navbar
 1. Converts into hamburger menu on smaller screens
 2. Toggle visibility using JavaScript
- Interactive Cards
 1. Hover effect with lift animation
 2. Smooth transitions for better UX
