# Digital Agency Portfolio Website

A responsive **Digital Agency Portfolio** built using **HTML, CSS, and JavaScript**.  
This project demonstrates modern web design techniques such as responsive layouts, micro-interactions, and dark mode functionality.

---

## Features

### Level 1 Requirements
- Responsive design for **Mobile and Desktop**
- **Navigation Bar**
  - Logo on the left
  - Navigation links on the right (Home, About, Services, Contact)
- **Hero Section**
  - Headline
  - Sub-headline
  - "Get Started" call-to-action button
- **Services Section**
  - 3 service cards displayed using **CSS Grid**
  - Example services: SEO, Web Development, Digital Marketing
- **Footer**
  - Copyright text
  - Social media icons

### Level 2 Requirements
- **Dark Mode Toggle**
  - Switch between light and dark themes
  - Implemented using JavaScript and CSS variables
- **Micro-Interactions**
  - Buttons change color and scale on hover
  - Service cards lift slightly when hovered
- **Sticky Navbar**
  - Navbar remains visible at the top while scrolling

---

## Technologies Used

- **HTML5** – Page structure
- **CSS3** – Styling and responsive design
- **Flexbox** – Navbar layout
- **CSS Grid** – Services section layout
- **JavaScript (Vanilla JS)** – Dark mode toggle and menu functionality
- **Font Awesome** – Social media icons
