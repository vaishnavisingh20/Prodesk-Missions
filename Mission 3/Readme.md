## Dev Detective – GitHub Profile Analyzer
Dev Detective is a web application that allows users to search GitHub profiles, analyze developer information, and compare two developers using Battle Mode.
It uses the GitHub REST API to fetch real-time data and display it in a clean, responsive UI.

---
## Technologies used
- HTML5 – Page structure
- CSS3 – Styling and layout
- JavaScript (ES6) – Application logic
- GitHub REST API – Fetching developer data

---
## Project Structure
- index.html
- styles.css
- script.js
- README.md
- Note: No Prompts.md is there
---
## How to run the project
- Step 1: Clone the repository
```
git clone https://github.com/yourusername/dev-detective.git
```
- Step 2: Open the project folder
- Step 3: Open index.html in your browser
---
## Example workflow:
- Enter a Github username
- Click Search
- The profile and repositories will appear
- Enable Battle mode to compare two developers
