## Dev Detective
---
Dev Detective is an advanced web application that allows users to search, analyze, and compare GitHub profiles in real-time.

It goes beyond basic profile viewing by introducing a unique Battle Mode 🥊, where two developers compete based on their GitHub stats.

---
## Features
---
- Profile Search
  Search any GitHub user by username
   Displays:
   * Avatar 
   * Name & Bio
   * Join Date 
   * Portfolio Link 
- Repository Insights
   Shows Top 5 Repositories
   Sorted by latest updates
   Clickable repo links
- Battle Mode (Unique Feature)
   Compare two GitHub users
   Automatically determines winner 
   Based on followers count
   Visual indicators:
   * Winner highlight
   * Loser highlight
   * “WINNER” badge
- Real-Time API Integration
   Uses GitHub REST API
   Async data fetching with error handling
- Loading & Error Handling
   Loading spinner animation
   User-friendly error messages
- Responsive UI
   Works across devices
   Clean dark-themed interface
---
## Tech Stack
- HTML5 – Structure
- CSS3 – Styling & animations
- JavaScript (Vanilla) – Logic & API handling
- GitHub API – Data source
