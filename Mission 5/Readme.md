## Kanban Task Board
---
The Kanban Task Board is a dynamic task management application inspired by tools like Trello.

Built using React.js, it allows users to create, manage, and track tasks across different stages of completion using an intuitive column-based layout.

---
## Features
- Core Functionality
  Add Tasks to "To Do" column
  Delete Tasks instantly
  Move Tasks between:
- To Do
- In Progress
- Done
---
## Advanced Features
- Edit tasks inline
- Priority System
  High Priority
  Medium Priority
  Low Priority
- Local Storage Persistence
  Tasks remain even after refresh
---
## Tech Stack
- React.js – UI & Component-based architecture
- Vite – Fast development environment
- JavaScript – Logic
- CSS3 – Styling
